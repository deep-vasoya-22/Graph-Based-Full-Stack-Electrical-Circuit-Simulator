A full-stack web-based circuit simulator built as part of a Data Structures & Algorithms project.

This project models electrical circuits using graph-based data structures and integrates a C++ simulation engine with a Node.js backend and JavaScript frontend.

Features

  Interactive circuit design interface

  Graph-based circuit representation

  Backend simulation engine in C++

  Node.js server integration

  Real-time component connectivity handling

  Modular architecture for scalability

System Architecture

  Frontend (JavaScript) -> Node.js Backend -> C++ Simulation Engine

The circuit is modeled as a graph where:

  Nodes represent junctions

  Edges represent electrical components

  This allows efficient traversal and connectivity analysis.

Tech Stack

  C++ – Core simulation logic

  Node.js – Backend server

  JavaScript – Frontend interaction

  HTML/CSS – UI

Concepts Used

  Graph Data Structures

  Adjacency Lists / Connectivity Modeling

  Modular System Design

  Backend-Frontend Integration
